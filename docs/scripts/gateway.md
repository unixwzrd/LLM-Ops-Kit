# gateway

**Created**: 2026-02-26
**Updated**: 2026-02-26

## Purpose

Manage OpenClaw gateway lifecycle.

## Syntax

```bash
~/bin/gateway [start|stop|restart|status]
```

## Examples

```bash
~/bin/gateway start
~/bin/gateway status
```

## Notes

Use `status` after `restart` to confirm healthy pid/log state.
