# Qwen3

**Created**: 2026-02-26
**Updated**: 2026-02-26

## Purpose

Run/control Qwen3 llama.cpp profile.

## Syntax

```bash
~/bin/Qwen3 [start|stop|restart|status|settings]
```

## Examples

```bash
~/bin/Qwen3 settings
~/bin/Qwen3 restart
```

## Notes

`settings` prints effective runtime parameters and template path.
